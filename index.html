Add a domain

After adding, run the audit script in Terminal to scan it. The new domain will show as "not yet audited" until then.

How the audit refreshes

The audit runs automatically on GitHub Actions every day at ~7am NZ time. No action needed from you.

Want to regenerate now?

Click the Regenerate data button on the dashboard — it opens the GitHub Actions page where you click Run workflow to kick off an immediate run (takes about 20 seconds).

Apply your changes

You've edited the dashboard locally. To make the audit script pick up these changes, write the new config to ~/Documents/audit-config.json.
